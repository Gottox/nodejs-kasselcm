  • Erstes Kickstarter Projekt mit >10mil $

  • B/W Display

Installation

mkdir ~/pebble-dev/
cd ~/pebble-dev/
tar zxf ~/Downloads/PebbleSDK-3.4.tar.gz
cd PebbleSDK-3.4
tar zxf \
   ~/Downloads/arm-cs-tools-$PLATTFORM.tar.gz
export PATH=$PATH:~/pebble-dev/PebbleSDK-3.4/bin

SDK

usage: pebble [-h] [--version]

  • new-project: Erstellt Projekt
  • build: Baut das Projekt
  • login: registriert sich bei Cloudpebble
  • install --emulator {aplite,basalt}: Startet Emulator mit Projekt
  • install --cloudpebble: Installiert auf der Uhr

pebble new-project helloworld

$ pebble new-project helloworld
